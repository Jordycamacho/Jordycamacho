    • KunturTattoo

    KunturTattoo es una innovadora aplicación web dirigida tanto a usuarios como a tatuadores. Permite a los tatuadores gestionar sus diseños y citas, mientras que los usuarios pueden explorar y filtrar diseños por categorías de una manera eficiente y organizada.

    Video Demostrativo

    Descripción General

    La plataforma fue creada con el objetivo de facilitar la gestión de tatuajes y citas para tatuadores y mejorar la experiencia de búsqueda para los usuarios. Utilizando una arquitectura moderna y robusta, se prioriza tanto la usabilidad como la seguridad en todas las áreas del proyecto.

    Características Principales

    • Exploración de Diseños: Los usuarios pueden navegar por diferentes categorías de tatuajes y aplicar filtros para encontrar el diseño perfecto.
    • Gestión de Citas: Los administradores pueden gestionar, crear, editar y eliminar citas de manera eficiente.
    • Notificaciones Automáticas: Sistema de envío de correos diarios para notificar al administrador sobre las citas pendientes.
    • Formulario de Contacto: Los usuarios pueden enviar mensajes personalizados al administrador con detalles específicos para solicitar tatuajes.

    Funcionalidades del Usuario

    Los usuarios de la plataforma pueden realizar diversas acciones dentro de KunturTattoo, enfocadas en la exploración de diseños y contacto con tatuadores.

    • Explorar diseños de tatuajes por categorías.
    • Filtrar los resultados según preferencias (estilo, tamaño, etc.).
    • Contactar con los tatuadores mediante un formulario que incluye correo electrónico, asunto, tamaño del tatuaje, parte del cuerpo y links de referencia.

    Funcionalidades del Administrador

    El administrador cuenta con una interfaz completa de gestión, con opciones que facilitan la creación y edición de contenidos, así como la supervisión de usuarios y citas.

    • Gestión de diseños: Crear, editar y eliminar diseños de tatuajes.
    • Gestión de categorías: Crear y eliminar categorías de tatuajes.
    • Gestión de usuarios: Revisar la lista de usuarios registrados.
    • Gestión de citas: Crear, editar, eliminar citas y recibir correos automáticos con recordatorios diarios sobre las citas pendientes.

    Seguridad y Testing

    La aplicación garantiza un control de acceso seguro mediante la implementación de Spring Security. Los usuarios están clasificados en roles (admin, user, guest), y los permisos están detallados para cada acción (crear, actualizar, eliminar, leer).

    Adicionalmente, el proyecto incluye un sistema de pruebas exhaustivas utilizando Mockito para asegurar que todas las funcionalidades críticas operan correctamente. Estas pruebas abarcan:

    • Controladores: Validación de los endpoints y respuesta de servicios REST.
    • Servicios: Verificación de la lógica de negocio.
    • Repositorios: Pruebas de consultas JPA y manejo de datos con MySQL.

    Tecnologías Utilizadas

    • Backend: Java, Spring Boot, Spring Security, Hibernate.
    • Frontend: HTML, CSS, JavaScript, Thymeleaf.
    • Base de Datos: MySQL.
    • Correo Electrónico: JavaMailSender para el envío automático de correos.
