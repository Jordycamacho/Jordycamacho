    • Ecommerce

    Ecommerce es una plataforma de compras en línea desarrollada con el objetivo de proporcionar una experiencia de compra fluida, segura y optimizada. El proyecto se enfoca en la gestión eficiente de productos y pedidos, brindando a los usuarios la posibilidad de explorar una variedad de artículos, realizar pagos seguros y recibir actualizaciones constantes sobre sus compras.

    Video Demostrativo

    Descripción General

    El desarrollo de este Ecommerce se centró en crear una plataforma robusta y escalable que permita a los usuarios registrarse, navegar entre productos, añadir artículos al carrito y completar compras de manera segura. Además, la plataforma cuenta con roles diferenciados para usuarios y administradores, asegurando una gestión eficiente del sitio.

    Características Principales

    • Exploración de Productos: Los usuarios pueden navegar por una amplia gama de productos categorizados para una fácil búsqueda.
    • Carrito de Compras: Los usuarios pueden agregar productos a su carrito, ver el total acumulado y proceder al pago.
    • Gestión de Productos: Los administradores tienen la capacidad de añadir, editar y eliminar productos de la plataforma.
    • Pagos Seguros: La plataforma utiliza la API de Stripe para procesar pagos de manera segura y eficiente.
    • Notificaciones Automáticas: Envío de correos electrónicos a los usuarios para confirmar el estado de sus pedidos.

    Funcionalidades del Usuario

    Los usuarios tienen acceso a una amplia gama de funcionalidades para mejorar su experiencia de compra en línea. Entre las principales características disponibles para los clientes se incluyen:

    • Registro y autenticación de usuarios con seguridad proporcionada por Spring Security.
    • Exploración de productos, con la posibilidad de filtrar por categorías o precios.
    • Gestión de carritos de compra, con actualización en tiempo real de los productos seleccionados.
    • Realización de pagos seguros mediante la integración de Stripe.
    • Recepción de correos electrónicos para confirmar compras y envíos.

    Funcionalidades del Administrador

    Los administradores cuentan con una interfaz dedicada para gestionar eficientemente la plataforma, manteniendo el control sobre los productos, las órdenes y los usuarios registrados.

    • Gestión completa de productos: Crear, editar y eliminar artículos.
    • Control de usuarios: Supervisar y administrar los roles de usuario y administrador.
    • Gestión de órdenes: Revisión y actualización de los pedidos realizados por los usuarios.
    • Envío de correos automáticos para mantener a los clientes informados sobre sus compras.

    Seguridad y Testing

    Para garantizar la seguridad de la plataforma, se implementaron roles específicos para usuarios y administradores utilizando Spring Security. Además, todas las transacciones de pago son manejadas de manera segura a través de la API de Stripe, protegiendo la información financiera de los clientes.

    El proyecto también incorpora pruebas automatizadas con Mockito y JUnit para asegurar que la plataforma funcione sin problemas. Estas pruebas verifican:

    • Controladores: Validación de los endpoints del Ecommerce y respuesta de los servicios REST.
    • Servicios: Verificación de la lógica de negocio y el manejo de productos, usuarios y órdenes.
    • Repositorios: Pruebas de consultas JPA para la gestión eficiente de datos en MySQL.

    Tecnologías Utilizadas

    • Backend: Java, Spring Boot, Hibernate, Spring Security.
    • Frontend: HTML, CSS, Bootstrap, Thymeleaf.
    • Base de Datos: MySQL.
    • Pagos Seguros: Integración con Stripe.
    • Correo Electrónico: JavaMailSender para el envío de notificaciones.
